API Versions

The Metasys REST API was first released with Metasys 10.0 and has seen many changes and additions since then. Please select the appropriate set of documentation for your release of Metasys. Hello Test PR #2.

  • Version 6 for Metasys 15 This is the second release of Metasys which has not changed the version number of the API. There are non-breaking changes and some newly deprecated operations which make this version different from the 14.1 version.
  • Version 6 for Metasys 14.1 This is the first release of Metasys which has not changed the version number of the API. There are a few non-breaking changes to the API that make this version 6 differ slightly from the version released with Metasys 14.
  • Version 6 for Metasys 14
  • Version 5 for Metasys 13 and 14
  • Version 4 for Metasys 12 and 13
  • Version 3 for Metasys 11 and 12
  • Version 2 for Metasys 10.1, 11 and 12
  • Version 1 for Metasys 10

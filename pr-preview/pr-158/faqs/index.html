FAQs

REST API

Are there any usage examples?

New

We have created a few examples as a means to show some use cases of how the API might be used. One of the examples is a data extraction utility which aggregates data from within and stores it locally for external analysis.

How do I get an object identifier?

New

All of the API endpoints accept GUIDs as identifiers for retrieving information for that object. These identifiers may be obtained by querying for a collection of the requested objects and in turn, the id property on each item will be what is used for subsequent requests in relation to that object.

Documentation

How do I contribute to this documentation?

New

The landing page is hosted via GitHub pages with the source located in the Server API Landing repo whereas the actual API documentation is housed in the Server API Docs repo. The latter uses Apiary to handle the rendering and publishing of the documentation.
